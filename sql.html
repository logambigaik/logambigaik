SQLBolt Practice Exercises

Lesson 1: SELECT Queries ▶
Description: Learn how to retrieve data from a table using SELECT statements.
SQL Query: 
SELECT name, age FROM students WHERE age > 20;
Output: 
+----------+-----+
| name     | age |
+----------+-----+
| Alice    | 21  |
| Johnson  | 23  |
+----------+-----+
Lesson 2: Filtering Rows ▶
Description: Use WHERE clauses to filter data.
SQL Query: 
SELECT * FROM employees WHERE department = 'Sales';
Output: 
+----+-----------+-------------+
| id | name      | department  |
+----+-----------+-------------+
| 1  | John Doe  | Sales       |
+----+-----------+-------------+
Find all the WALL-* movies ▶
Description: Retrieve all movies with titles that start with "WALL-".
SQL Query: 
SELECT * FROM movies WHERE Title LIKE 'WALL-%';
List the next five Pixar movies alphabetically ▶
Description: Display the 6th to 10th Pixar movies sorted by title.
SQL Query: 
SELECT Title, Year FROM movies
ORDER BY Title ASC
LIMIT 5 OFFSET 5;
Find the domestic and international sales for each movie ▶
Description: Combine data from movies and box office to show sales figures.
SQL Query: 
SELECT Title, Domestic_sales, International_sales 
FROM movies 
JOIN Boxoffice ON movies.id = Boxoffice.movie_id;
Show movies that performed better internationally ▶
Description: List movies where international sales exceeded domestic sales.
SQL Query: 
SELECT Title, Domestic_sales, International_sales 
FROM movies 
JOIN Boxoffice ON movies.id = Boxoffice.movie_id 
WHERE International_sales > Domestic_sales;
List all movies by ratings (highest to lowest) ▶
Description: Join with box office data and sort all movies by rating.
SQL Query: 
SELECT Title, Rating 
FROM movies 
JOIN Boxoffice ON movies.id = Boxoffice.movie_id 
ORDER BY Rating DESC;
