Tools & Techniques: Seaborn - Visualizing the Data ▶
Histogram - All Numeric Columns
Code: 
      import matplotlib.pyplot as plt
      import seaborn as sns
      df.hist(figsize=(10,8))
      plt.show()
Description:
  • Generates histograms for all numeric columns in the DataFrame.
  • hist() plots distributions; figsize adjusts plot size.
Output:
Histogram - Selected Columns
Code: 
      df[["IMF_Estimate", "UN_Estimate", "WorldBank_Estimate"]].hist(bins=3, figsize=(12,9))
      plt.show()
Description:
  • Plots histograms for specific columns: IMF, UN, and World Bank estimates.
  • bins=3 defines the number of intervals; figsize adjusts layout size.
Output:
Heatmap - Basic
Code: 
      corr = df[["IMF_Estimate", "UN_Estimate", "WorldBank_Estimate"]].corr()
      plt.figure(figsize=(9,6))
      sns.heatmap(corr)
      plt.show()
Description:
  • Computes correlation matrix for selected estimates.
  • sns.heatmap() visualizes numeric relationships with color intensity.
Output:
Heatmap - With Annotations & Color
Code: 
      corr = df[["IMF_Estimate", "UN_Estimate", "WorldBank_Estimate"]].corr()
      plt.figure(figsize=(9,6))
      sns.heatmap(corr, annot=True, fmt=".2f", cmap='GnBu', annot_kws={"size": 12})
      plt.show()
Description:
  • annot=True: Adds numeric values in the heatmap cells.
  • fmt=".2f": Displays values with 2 decimal places.
  • cmap='GnBu': Sets the color palette.
  • annot_kws={"size": 12}: Adjusts annotation font size.
